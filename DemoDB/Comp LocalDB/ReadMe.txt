	Простая демострационныя база для учёта компьютеров.
Имеется вариант как FB/IB для ADO, IB так и Paradox для BDE
платформ.

Для работы с локальной базой поместите файл DCLRun.exe из
папки BDE в этот каталог, при этом у Вас должен быть установлен
BDE.

Для запуска на сетевой СУБД, выберите подходящий тип платформы
для Вашего подключения, прописав необходимые параметры в файле
DCL.ini.
